### Overview

NG Release Bell is a self-hosted release notification service that tracks releases across multiple providers: GitHub, GitLab, Gitea/Codeberg, NPM, PyPI, Docker Hub, Quay, GitHub Container Registry, and SourceForge.

Per-project notification settings include email frequency (instant, hourly, daily, weekly), pre-release exclusion, re-release suppression, and regex-based version filtering with live preview.

All notifications are sent out via email.
