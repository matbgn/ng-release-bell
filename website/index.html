Features




Multi-Provider

Track releases across GitHub, GitLab, Gitea/Codeberg, NPM, PyPI, Docker Hub, Quay, GHCR, and SourceForge.

Self-Hosted

Run on your own server. Only requires Node.js, MySQL and an email server.

Email Notifications

Receive instant, hourly, daily, or weekly email digests for new releases.

Per-Project Settings

Configure email frequency, pre-release exclusion, re-release suppression, and regex version filtering per project.
